INTRODUCTION
------------

QED42 module is developed for creating custom block and custom taxonomy Category.

Simply put the modules under modules/custom directory and just enabled the module.

REQUIREMENTS
------------

No special requirements.


RECOMMENDED MODULES
-------------------

No special recommended modules required but it will look better if you install and enabled Admin Toolbar . 
* Admin Toolbar Extra Tools (https://www.drupal.org/project/admin_toolbar):
   Provides article content type qed42 category field in  manage form display page or actions (eg. Flushing caches)
   that are not generated by Drupal core.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.
   Download module and compress/extract then rename (Qed42-master to Qed42).


CONFIGURATION
-------------
Add Block:- Go to structure->block layout->choose anyone region ->search Qed42 block and set visibility
 Content type Article check box select and save block.
Content Type Configuration:- Go to structure->content type-> select article content type and got to manage
 form display drug and drop for Qed42 Category field enable.
Note:- All cases test for add different users and add different type category article content for testing purpose.
